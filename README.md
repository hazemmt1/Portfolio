# Hazem AlTaamneh - Architecture Portfolio

A modern, responsive portfolio website built with **React**, **Vite**, and **Tailwind CSS**. Deployed on **GitHub Pages** with **EmailJS** contact form integration.

## **Quick Start**

### **Install & Run Locally**
```bash
npm install
npm run dev
```
Visit: `http://localhost:5173`

### **Build for Production**
```bash
npm run build
```

### **Deploy to GitHub Pages**
```bash
git add .
git commit -m "your message"
git push
```
Auto-deploys in 2-3 minutes. View at: `https://hazemmt1.github.io/Portfolio/`


## **Environment Variables**

Create `.env.local` with:
```
VITE_EMAILJS_PUBLIC_KEY=
VITE_EMAILJS_SERVICE_ID=
VITE_EMAILJS_TEMPLATE_ID=
```
